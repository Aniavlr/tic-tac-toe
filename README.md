# Tic-Tac-Toe Game with Firebase


A modern, full-featured **Tic-Tac-Toe** web application built with **React** and **Firebase**. Features authentication, real-time leaderboard, game history, user profiles, and bot opponent.

<!-- Live Demo: [https://your-app-url.firebaseapp.com](https://your-app-url.firebaseapp.com) *(update after deploy)* -->

## Features

- Email/password authentication (Firebase Auth)
- Play against an AI bot with realistic thinking delay
- Global leaderboard (top 10 players by score)
- Personal game history with date and result
- User profile with full statistics (wins, losses, draws, score)
- Responsive & beautiful UI with gradients, animations, and hover effects
- All game results saved in real time (Firestore)
- Loading states

**Scoring system:**
- Win → +10 points
- Draw → +5 points
- Loss → -5 points

## Tech Stack

- **Frontend**: React (functional components + hooks)
- **Backend**: Firebase (Authentication + Firestore)
- **Styling**: Pure CSS (no frameworks) — fully custom design
- **Hosting**: Firebase Hosting
- **State Management**: React hooks (`useState`, `useEffect`)
